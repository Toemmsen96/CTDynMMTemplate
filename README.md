# CTDynamicModMenu Template
This is an Example Template for a BepInEx Mod using CTDynamicModMenu.

## Installing the Template / Usage
1. Clone this Git repository
2. run ``` dotnet new install .  ``` 
3. Create a new project by using dotnet new ``` dotnet new ctdynmmplugin ```

## DISCLAIMER
This Template is currently only for the Mono Version of the Menu. The IL2CPP / BepInEx 6 Version does not work with this.
